# Community Agent Catalog

An open, community-driven catalog of AI agents for architects, managers, and practitioners.

## Structure

- agents/        → Individual agents
- tools/         → Reusable tools
- templates/     → Starter templates
- runtime/       → CLI and runtime code
- docs/          → Contribution and standards
- catalog/       → Generated catalog metadata

## Goal

Create a uniform, open, and reusable ecosystem of AI agents that anyone can build, run, and share.
